SPACE AWARENESS

Real-time orbital tracking of satellites, space stations, and orbital debris using client-side SGP4 propagation.

DATA SOURCES

The GEOINT.WORLD Space Data module performs orbital propagation using high-precision TLE datasets.

CELESTRAK (NORAD)

Primary source for Two-Line Element sets (TLEs) describing the orbit of Earth-orbiting objects.

Use the ⚙ icon next to the SPACE DATA layer in the application to access the advanced satellite filter menu.

ADVANCED FILTERS

The Satellite Filter panel provides granular control over which constellations are displayed.

NAVIGATION

  • GPS (USA)
  • GLONASS (Russia)
  • GALILEO (EU)
  • BEIDOU (China)

COMMUNICATION

  • STARLINK (SpaceX)
  • ONEWEB
  • IRIDIUM

OBSERVATION

  • WEATHER (GOES/METEOSAT)
  • NOAA
  • PLANET
  • CUBESATS

REFRESH

Control the automatic data refresh interval. Available options:

  • 30s — Refresh every 30 seconds
  • 1m — Refresh every 1 minute
  • 5m — Refresh every 5 minutes

Use the HIDE button to minimize the filter panel and maximize map visibility.
