EARTHQUAKES

EARTHQUAKE DATA

Real-time global seismic activity monitoring using USGS earthquake feeds.

DATA SOURCES

The GEOINT.WORLD Earthquake module utilizes real-time seismic data feeds provided by the U.S. Geological Survey.

USGS (U.S. GEOLOGICAL SURVEY)

Official source for global earthquake monitoring with near-instantaneous detection.

FILTERS & CLASSIFICATIONS

Use the Earthquake Settings Panel to control which seismic events are displayed on the map.

TIMEFRAME

Select the time window for earthquake data:

  • 1H — Past Hour
  • 1D — Past 24 Hours (default)
  • 7D — Past 7 Days
  • 30D — Past 30 Days

MAGNITUDE FILTER

Control the minimum magnitude threshold for displayed earthquakes. Options include M1.0+, M2.5+ (default), M4.5+, and ALL.

MAGNITUDE COLOR CODING

Earthquakes are color-coded by magnitude for rapid severity assessment:

  • M2.5 - M3.9 — Minor (Yellow)
  • M4.0 - M4.9 — Light (Orange)
  • M5.0 - M5.9 — Moderate (Red)
  • M6.0 - M6.9 — Strong (Dark Red)
  • M7.0+ — Major/Great (Deep Red)
Earthquakes above M5.0 display a pulsing ring effect on the map to highlight significant seismic events.
