FILTERS & CLASSIFICATIONS

Use the Flight Control Panel to filter visible aircraft by category.

  • VERIFIED TRAFFIC — Routine commercial flights (Airliners, Cargo, GA).
  • MILITARY — Air Force, Navy, and State-operated aircraft.
  • AIRCRAFTS WITH PRIVATE PIA ADDRESSES — Jets using rolling identifiers for anonymity.
  • AIRCRAFTS LIMITING DATA DISPLAYED — Owners blocking public tracking (LADD).
  • NO CALLSIGN — Transmitting position but NO Identity.
  • NO SQUAWK CODE — Aircraft not broadcasting a squawk code.
  • ALERT — Emergency squawk codes or SPI.
Use the i icon next to the FLIGHT DATA layer in the application to access the real-time legend and advanced descriptions.

EMERGENCY SQUAWK CODES

The system automatically flags aircraft broadcasting international emergency codes as ALERT.

CODE MEANING
7700 GENERAL EMERGENCY. Mechanical failure, medical issue, or distress.
7600 RADIO FAILURE. Loss of communication with ATC.
7500 HIJACKING. Unlawful interference.
SPI SPECIAL POSITION IDENT. Pilot pressed 'IDENT' button—also triggers ALERT status on the map.

ADVANCED FILTERS

The Flight Filter panel provides additional controls for precise aircraft filtering.

SQUAWK FILTER

Use the search field to filter flights by specific squawk codes. Enter a squawk code (e.g., 7700, 1200) to display only aircraft broadcasting that code.

MIN ALTITUDE

Use the slider to set a minimum altitude threshold. Aircraft below this altitude will be hidden from the map. Default is 0 meters. Adjust the slider to focus on high-altitude flights or filter out ground traffic.

REFRESH

Control the automatic data refresh interval. Available options:

  • 1m — Refresh every 1 minute (default)

Use the HIDE button to minimize the filter panel and maximize map visibility.
