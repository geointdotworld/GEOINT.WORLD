GLOBAL EVENTS & NEWS

Real-time monitoring of global news events using GDELT's machine intelligence platform.

DATA SOURCES

The GEOINT.WORLD News module leverages the GDELT Project's monitoring system to track and analyze global events in real-time.

THE GDELT PROJECT

Monitors worldwide news media in over 100 languages, identifying events, people, organizations, and locations in near real-time.

FILTERS & CONTROLS

The GDELT News Filter provides granular control over which events appear in your feed.

KEYWORD FILTER

Add positive keywords to track specific event types. The system fetches news matching any of your selected keywords.

Default Keywords:

  • MILITARY — Military operations and deployments
  • CONFLICT — Armed conflicts and disputes
  • ATTACK — Attacks and violent incidents

To add custom keywords, type in the keyword input field and press Enter.

NEGATIVE KEYWORDS (EXCLUDE)

Negative keywords filter out noise by excluding unwanted topics. Headlines matching any negative keyword will be removed from results.

Common Exclusions:

  • -JAIL, -COURT, -TRIAL — Legal proceedings
  • -FINANCE, -ECONOMY — Financial news
  • -SUMMIT — Diplomatic summits
  • -ENTERTAINMENT, -SPORTS, -MUSIC, -FASHION — Lifestyle content

TIMEFRAME

Select the time window for news data retrieval:

  • 1H — Past Hour (default)
  • 12H — Past 12 Hours
  • 24H — Past 24 Hours
Click "FETCH LATEST DATA" to manually refresh the news feed with current filters applied.

TABLE VIEW

The news interface presents results in a sortable table with the following columns:

  • NEWS HEADLINE — Event description with result count
  • KEYWORD — Matched filter tag (e.g., MILITARY)
  • SOURCE — News outlet domain
  • COUNTRY — Event location
  • DATE — Timestamp (sortable)

Use the search bar to filter headlines by text. Click column headers to sort results.
