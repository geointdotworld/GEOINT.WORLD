PREDICTION MARKETS

Real-time geopolitical forecasting using decentralized prediction markets.

DATA SOURCES

The GEOINT.WORLD Prediction module tracks high-volume unresolved markets from Polymarket, the world's largest decentralized prediction platform.

POLYMARKET

Decentralized information markets providing real-time probabilities for global events using blockchain technology.

FILTERS & CONTROLS

The Polymarket Filter provides advanced tools to focus on relevant prediction markets.

KEYWORD FILTER

Add keywords to track specific geopolitical topics. The system fetches markets matching any of your selected keywords.

Default Keywords:

  • STRIKE — Military strikes and operations
  • TRUMP — Trump-related events
  • ATTACK — Attack predictions
  • IRAN — Iran-related markets
  • PUTIN — Putin/Russia events
  • ISRAEL — Israel-related predictions
  • KOREA — Korean peninsula events

To add custom keywords, type in the keyword input field and press Space or Comma.

MINIMUM CHANCE %

Use the slider to filter markets by their current probability. Default is 30%. Only markets with a "Yes" probability at or above this threshold will be displayed.

MINIMUM 24H VOLUME

Set a minimum 24-hour trading volume (in USD) to focus on high-liquidity markets. Default is $500. Markets with lower daily volume are excluded.

Use the "SET" button to apply your custom volume threshold.

GROUP BY MARKET

Toggle this option to group related prediction questions under the same parent market, reducing redundancy and improving clarity when multiple outcomes are being tracked for the same event.

TABLE VIEW

Prediction markets are displayed in a sortable table with the following columns:

  • MARKET QUESTION — Prediction question with result count
  • NEWEST — Latest market creation date
  • 24H VOL % — 24-hour volume as percentage of total volume
  • TOTAL VOL — Lifetime trading volume
  • CHANCE — Current "Yes" probability (color-coded)

Use the search bar to filter markets by text. Click column headers to sort results. The "FILTERED" button shows the total number of markets matching your active filters.
